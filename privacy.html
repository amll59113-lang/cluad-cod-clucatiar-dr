Privacy Policy

Last updated: April 28, 2026

AffiliateTrackerPro is a personal, single-user affiliate marketing dashboard. It does not collect, store, or share any personal data from third parties. All data processing occurs locally in the user's browser.

1. Overview

AffiliateTrackerPro ("the Application") is a personal performance dashboard hosted at affiliatetrackerpro.com. It is built and operated by an individual affiliate marketer for the sole purpose of tracking personal advertising campaigns and affiliate earnings.

2. Data We Collect

The Application connects to the following third-party APIs using credentials provided by the owner:

  • TikTok Marketing API — to retrieve daily ad spend data from the owner's own TikTok Ads account
  • Dr.Cash Affiliate Network API — to retrieve commission and lead statistics from the owner's affiliate account

All API credentials (access tokens, advertiser IDs) are stored locally in the browser's localStorage and are never transmitted to any external server operated by this application.

3. How We Use Data

Data retrieved from third-party APIs is used exclusively to:

  • Display personal ad spend, profit, and ROI metrics
  • Calculate campaign performance statistics
  • Show lead tracking data (approved, hold, rejected)

No data is used for advertising, profiling, or any purpose beyond personal performance tracking.

4. Data Storage

All data is stored locally in the user's browser using localStorage. No data is sent to, stored on, or processed by any server operated by AffiliateTrackerPro. The application has no backend server or database.

5. Third-Party Services

The Application integrates with the following third-party services. Users are subject to their respective privacy policies:

  • TikTok for Business — TikTok Marketing API
  • Dr.Cash — Affiliate network API
  • Google Fonts — Font rendering (Inter typeface)

6. Cookies

The Application uses browser localStorage only to store authentication state and user preferences. No tracking cookies are used.

7. User Access

The Application is password-protected and intended for use by the owner only. No public registration or user accounts are available.

8. Children's Privacy

The Application is not intended for use by individuals under the age of 18. No data from minors is knowingly collected.

9. Changes to This Policy

This Privacy Policy may be updated from time to time. The "Last updated" date at the top of this page reflects the most recent revision.
